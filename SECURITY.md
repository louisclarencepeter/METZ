# Security Policy

## Reporting a vulnerability

If you discover a security issue in this site or repository, please report it
privately rather than opening a public GitHub issue.

- Email: metz_engineering@yahoo.com
- Subject line: "Security report — METZ website"

Please include:

- A description of the issue and where you observed it
- Steps to reproduce, including URLs, browser, and any sample payloads
- The impact you believe the issue has

We will acknowledge the report and provide a status update within 7 days.

## Supported versions

Only the version of the site currently deployed from the `main` branch is
supported. Older builds are not patched.

## Scope

In scope:

- The deployed METZ Engineering website
- Source code in this repository

Out of scope:

- Findings that require physical access to a user's device
- Denial-of-service testing against the production site
- Automated scanner output without a working proof of concept
